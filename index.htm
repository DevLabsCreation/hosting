Hosting a Website on GitHub

Step 1: Create a GitHub Account

Go to GitHub.com and click on "Sign up" in the top right corner.

Fill out the registration form with your details and click on "Create account."

Verify your email address by clicking on the link sent to you by GitHub.

Step 2: Create a New Repository

Log in to your GitHub account and click on the "+" icon in the top right corner.

Select "New repository" and fill out the repository name and description.

Choose "Public" and click on "Create repository."

Step 3: Upload Your Website Files

Go to your repository and click on "Add file" > "Upload files.". Ensure you upload the HTML file and the pictures as well.

Commit the changes

Step 4: Configure GitHub Pages

Go to your repository settings and scroll down to "GitHub Pages."

Select the branch you want to use for GitHub Pages (usually "main" or "master").

Save the changes and wait for GitHub to build your website.

Step 5: Visit Your Website

Once GitHub Pages is configured, you can visit your website at https://your-username.github.io/your-repo-name.

Replace "your-username" and "your-repo-name" with your actual GitHub username and repository name.
